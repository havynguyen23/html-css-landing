# HTML CSS Landing

A landing page built from scratch using only HTML and CSS — no frameworks, no libraries. The goal is to apply HTML and CSS fundamentals in a real project, using Vite as the build tool.

## Prerequisites

- [Node.js](https://nodejs.org/) v24.14.1 (specified in `.nvmrc`)
- [pnpm](https://pnpm.io/) v10.21.0

If you use [nvm](https://github.com/nvm-sh/nvm), run the following to switch to the correct Node version:

```bash
nvm use
```

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Preview production build

```bash
pnpm preview
```
